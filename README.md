# STIVO DEVX STUDIO | Solutions Tech & Cybersécurité

## Description

STIVO DEVX STUDIO est un studio d'innovation technologique spécialisé dans le **développement**, la **cybersécurité** et la création de **solutions digitales avancées**. Notre objectif est de rassembler une communauté de développeurs, experts en sécurité informatique et innovateurs passionnés pour construire l'écosystème tech de demain.

---

## Fonctionnalités Principales

- **Navigation fluide et responsive** adaptée à tous les écrans.
- **Section Hero** immersive présentant le studio et sa mission.
- **Plateformes communautaires** : WhatsApp, Telegram, GitHub, YouTube, X (Twitter), Instagram, Facebook.
- **Stats section** avec les chiffres clés : membres, projets open source, plateformes actives et support 24/7.
- **À propos** avec valeurs et mission de STIVO DEVX.
- **Footer complet** avec liens rapides, plateformes principales et contact/support.
- **Animations au scroll** pour une expérience interactive.
- **Effet parallaxe subtil** sur la section Hero.
- **Design moderne** avec gradient, glass effect et couleurs professionnelles.

---

## Tech Stack

- **HTML5**
- **CSS3** (variables CSS, flexbox, grid, animations)
- **JavaScript** (smooth scroll, animations Intersection Observer, effet parallaxe)
